Sessão 01: Mudar texto de um elemento

Temos nessa sessão um botão e precisamos mudar o texto quando clicarmos no botão. Você pode se basear nessa estrutura para adicionar um evento a um elemento. 

          [elemento].addEventListener(["nomeDoEvento"], [função]);

          // Siga esse exemplo para essa sessão:

          document.getElementById("idDoBotão").addEventListener("click", function () {
            document.getElementById("idDoTexto").innerText = "Novo texto";
          });

Sessão 03: Escondendo elementos

Na sessão passada vimos que podemos modificar classes utilizando o atributo className, porém também podemos utilizar o classList para alternar se queremos adicionar e remover uma classe por exemplo, utilizando o .toggle(). Você pode se basear nessa estrutura para adicionar um evento a um elemento. 

          [elemento].addEventListener(["nomeDoEvento"], [função]);

          // Siga esse exemplo para essa sessão:

          document.getElementById("idDoBotão").addEventListener("click", function () {
            document.getElementById("idDoTexto").classList.toggle("escondido");
          });

          // Adicione no seu css essa classe

          .escondido {
            display: none;
          }

Sessão 04: Utilizando outros eventos

Não existe somente o evento de click, também temos o evento de mouseover para quando o mouse passa por cima do elemento, e mouseleave quando o mouse sai do elemento. Sabendo disso, e sabendo que você pode adicionar qualquer número de eventos a um mesmo elemento eu quero que você adicione e remova a classe escondido do texto. Você pode se basear nessa estrutura para adicionar um evento a um elemento. 

          [elemento].addEventListener(["nomeDoEvento"], [função]);

          // Siga esse exemplo para essa sessão:

          document.getElementById("idDoBotão").addEventListener("mouseover", function () {
            document.getElementById("idDoTexto").className = "escondido";
          });

          document.getElementById("idDoBotão").addEventListener("mouseleave", function () {
            document.getElementById("idDoTexto").className = "";
          });

          // Adicione no seu css essa classe caso não tenha ainda.

          .escondido {
            display: none;
          }

Me esconda
